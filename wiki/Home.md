## Project Summary

The system replaces manual textile quotation methods with a web application that:

1. Registers customers via an enquiry form
2. Maintains a product catalogue with base prices
3. Generates multi-item quotations with GST calculated automatically
4. Stores a price snapshot at time of quote so old quotations are never affected by price changes
